# Könyv projekt – Használati útmutató

Ez a projekt egy könyv anyagát tartalmazza, amely **Markdown** formátumban készült.  
A tartalom **Obsidian alkalmazásban** olvasható és kezelhető a legjobban, de **Visual Studio Code-ban (VS Code)** is teljes mértékben használható.

Az alábbi útmutató végigvezet mindkét lehetőségen.

---

## Ajánlott megnyitás: Obsidian

Az Obsidian kifejezetten Markdown-alapú jegyzetekhez és összekapcsolt szövegekhez készült, ezért ebben az alkalmazásban a könyv szerkezete, linkjei és formázása a legátláthatóbb.

### 1. Obsidian telepítése

1. Nyisd meg a hivatalos oldalt: [https://obsidian.md](https://obsidian.md)
    
2. Töltsd le az operációs rendszerednek megfelelő verziót
    
3. Telepítsd az alkalmazást
    

### 2. A projekt megnyitása Obsidianban

1. Indítsd el az Obsidian alkalmazást
    
2. A kezdőképernyőn válaszd az **“Open folder as vault”** lehetőséget
    
3. Tallózd ki a könyv projekt **gyökérmappáját**
    
4. Nyisd meg a mappát
    

### 3. Olvasás és navigáció

- A fájlok bal oldalon, listában jelennek meg
    
- A belső linkek (`[[ ]]`) kattinthatók
    
- A fejezetek és hivatkozások gyorsan bejárhatók
    
- A Markdown formázás (címek, listák, kiemelések) optimálisan jelenik meg
    

---

## Alternatív megnyitás: Visual Studio Code

Ha nem szeretnél Obsidian-t használni, a projekt **VS Code-ban is működik**, mivel sima Markdown fájlokat tartalmaz.

### 1. Visual Studio Code telepítése

1. Nyisd meg: [https://code.visualstudio.com](https://code.visualstudio.com)
    
2. Töltsd le és telepítsd az alkalmazást
    

### 2. A projekt megnyitása VS Code-ban

1. Indítsd el a VS Code-ot
    
2. Válaszd a **File → Open Folder** menüpontot
    
3. Nyisd meg a könyv projekt mappáját
    

### 3. Markdown előnézet használata

- Nyiss meg egy `.md` fájlt
    
- Kattints jobb gombbal, majd **“Open Preview”**, vagy használd:
    
    - **Windows / Linux:** `Ctrl + Shift + V`
        
    - **macOS:** `Cmd + Shift + V`
        

Megjegyzés:  
VS Code-ban a belső Obsidian-linkek és a könyvszerű navigáció kevésbé kényelmes, de a tartalom olvasható és szerkeszthető.
